# Урок 7. Печатные формы и экспорт

**Длительность:** 3 ак. ч (1 теория + 2 практики)
**Пререк:** уроки 1–6.

## Цель урока

После урока обучающийся:
- Знает все стандартные формы сметной документации в РК и для чего каждая
- Корректно формирует формы из любой из трёх программ
- Экспортирует смету в форматах Excel/PDF и в формате обмена между программами

## Теория (45 мин)

### 7.1. Стандартные формы

| Форма | Назначение | Когда |
|---|---|---|
| **Форма №1 (ЛСР)** | Локальная сметная расчёт | На стадии П, Р; обновляется при изменениях |
| **Форма №2 (ОС)** | Объектная смета | На стадии П, Р по каждому объекту |
| **Форма №3 (ССР)** | Сводный сметный расчёт | На стадии П по стройке в целом |
| **Ведомость объёмов работ (ВОР)** | Обоснование объёмов | Прилагается к ЛСР |
| **Ведомость ресурсов (ВР)** | Перечень всех ресурсов и их стоимости | Прилагается к ЛСР, обязательна для ресурсного метода |
| **Пояснительная записка** | Описание примененных норм, индексов, обоснований | К комплекту сметной документации |
| **КС-2** | Акт о приёмке выполненных работ | На стадии производства работ (изучается на уровне 3) |
| **КС-3** | Справка о стоимости выполненных работ и затрат | Та же стадия |
| **КС-6а** | Журнал учёта выполненных работ | Та же стадия |

> Конкретные обозначения и нумерация форм — по актуальной редакции методических документов РК. На данном уровне (Пользователь) изучаются формы 1, 2, 3, ВОР, ВР, ПЗ.

### 7.2. Содержимое каждой формы

#### Форма №1 (ЛСР)
- Шапка: реквизиты, базисный уровень, итоговая стоимость
- Таблица позиций с разделами
- Начисления (НР, СП, лимитированные)
- Итоги

#### Форма №2 (ОС)
- Шапка с указанием объекта
- Таблица из всех ЛСР по этому объекту
- Сумма с лимитированными по объекту

#### Форма №3 (ССР)
- Заголовок стройки
- 12 глав (см. урок 4.2 уровня "Проектировщик")
- Резерв непредвиденных, НДС, итог

#### ВОР
- Таблица: позиция → формула → объём → ссылка на чертёж

#### ВР
- Таблица всех ресурсов сметы (труд, ЭМ, материалы) с количествами и ценами
- Особенно важна для ресурсного метода

### 7.3. Форматы экспорта

| Формат | Когда |
|---|---|
| **PDF** | Для печати, для подписания, для сдачи в экспертизу |
| **Excel (xlsx)** | Для работы с цифрами заказчика, для проверок |
| **Внутренний формат программы** (`.abc4`, `.smeta`, `.sana`) | Для дальнейшей работы в той же программе |
| **АРПС** (формат обмена сметами в РК) | Для обмена между разными программами и с экспертизой |

> АРПС — стандартизированный текстовый формат. Не все программы экспортируют идеально, иногда требуется ручная коррекция. Уточнить актуальность и распространённость на момент курса.

### 7.4. Печать и оформление

Перед печатью / экспортом проверить:
- Заполнен титул (объект, кто составил, кто проверил)
- Нумерация страниц
- Сноски в конце о применяемых нормативных документах
- Подписи: ставятся вручную после печати

В форматах документов экспертизы — обычно заверенная электронная копия с цифровой подписью.

## Параллельная практика (75 мин)

### Задание

Используя учебную смету (доделанную в уроках 1–6):

**Шаг 1:** в каждой из трёх программ сформировать:
- Форму №1 (ЛСР)
- ВОР
- ВР

Сохранить как PDF.

**Шаг 2:** сравнить три PDF-файла:
- Все ли разделы на месте?
- Совпадают ли итоги до копейки?
- Что отличается во вёрстке?

**Шаг 3:** экспортировать смету в формат АРПС из АВС-4. Импортировать в Смета РК. Проверить, что ничего не потерялось.

**Шаг 4:** экспортировать смету в Excel из всех трёх программ. Открыть три файла рядом, сравнить, найти расхождения (их быть не должно).

### Контроль

Преподаватель проверяет:
- Все формы сформированы корректно
- Импорт АРПС между программами прошёл без потерь
- Excel-выгрузки идентичны по цифрам

## Самопроверка (6 вопросов)

1. Что такое форма №1, №2, №3 — назначение каждой?
2. Что должно быть в ВОР?
3. Зачем нужна ведомость ресурсов?
4. В каком формате удобнее всего отдать смету заказчику для проверки в Excel?
5. Что такое АРПС и зачем он нужен?
6. Может ли вёрстка форм отличаться у разных программ при одинаковых цифрах?

## Домашнее задание

Сформировать полный комплект "форма №1 + ВОР + ВР + пояснительная записка" по своей учебной смете в одной программе на выбор. Сдать единым PDF-файлом.
