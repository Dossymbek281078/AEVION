# Методичка уровня 2 "Пользователь"

Это **образцовая методичка**, показывающая глубину проработки одного уровня. По её формату делаются методички остальных уровней.

## Состав методички

- [`урок-01-структура-лср.md`](./урок-01-структура-лср.md) — структура локальной сметы
- [`урок-02-поиск-расценок.md`](./урок-02-поиск-расценок.md) — поиск и применение расценок
- [`урок-03-объёмы.md`](./урок-03-объёмы.md) — подсчёт объёмов работ
- [`урок-04-нр-сп.md`](./урок-04-нр-сп.md) — накладные расходы и сметная прибыль
- [`урок-05-индексы.md`](./урок-05-индексы.md) — индексы пересчёта
- [`урок-06-материалы.md`](./урок-06-материалы.md) — индивидуальные цены материалов
- [`урок-07-формы.md`](./урок-07-формы.md) — печатные формы
- [`урок-08-сравнение-программ.md`](./урок-08-сравнение-программ.md) — одна задача в трёх программах

## Формат каждого урока

Каждый урок содержит:

1. **Цель урока** — что обучающийся будет уметь после
2. **Теоретическая часть** — нормативка + методика
3. **Демонстрация** — пошаговый разбор с указанием места под скриншот
4. **Параллельная практика** — одинаковая задача в АВС-4 / Смета РК / Сана
5. **Самопроверка** — 5–10 контрольных вопросов
6. **Домашнее задание** — задача для самостоятельного выполнения
7. **Ссылки на нормативку** — конкретные пункты СН РК

## Что нужно подготовить отдельно для каждого урока

> Это работа методиста-преподавателя, не самой методички:

- **Скриншоты** в трёх программах (по 5–8 на урок)
- **Видео-демонстрации** (5–10 минут на урок)
- **Эталонные файлы** (`.abc4`, `.smeta`, `.sana`) для сверки
- **PDF-формы** с подсветкой ключевых полей
- **Чертежи** для практических задач

В тексте методички места под эти артефакты помечены: `[СКРИНШОТ: ...]`, `[ВИДЕО: ...]`, `[ФАЙЛ: ...]`.
