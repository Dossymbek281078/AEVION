# Методичка уровня 5 "Эксперт"

**Длительность:** 80 ч (24 теории + 24 практики + 32 на кейс)
**Цель:** обучающийся проводит экспертизу сметной документации, выявляет завышения, защищает свои сметы, оппонирует внешним экспертам.

> **Внимание:** этот уровень не делает обучающегося государственным экспертом — этот статус устанавливается отдельно по законодательству. Уровень готовит **внутреннего эксперта** организации (заказчика, генподрядчика, проектной), способного качественно проверять сметы.
