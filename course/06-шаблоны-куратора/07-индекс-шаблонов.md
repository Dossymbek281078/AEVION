## Принципы оформления шаблонов

1. **Единый формат**: каждый шаблон — отдельный markdown-файл
2. **Структура**: реквизиты → содержание → подписи
3. **Заполняемые поля** обозначаются `___` или `[пояснение]`
4. **Примеры** — в виде блоков с типовым текстом
5. **Привязка к нормативке** — где применимо, ссылка на пункт СН РК
6. **Языковая адаптация** — при необходимости готовится казахская версия

## Использование

- Куратор раздаёт шаблоны обучающимся в **электронной форме** (PDF / Word / прямая ссылка)
- Обучающийся **заполняет** шаблон в реальной работе
- Куратор **проверяет** заполнение по соответствующей рубрике / чек-листу
- Заполненные шаблоны **сохраняются** в портфолио обучающегося
