# Урок 1. Состав сметного раздела проекта

**Длительность:** 4 ак. ч (2 теории + 2 практики)
**Пререк:** уровень 3 завершён.

## Цель

После урока обучающийся:
- Знает полный перечень документов сметного раздела проекта
- Понимает иерархию и взаимосвязи (ЛСР → ОС → ССР → ВР → ПЗ)
- Видит, кто что подписывает и в каком порядке

## Теория (90 мин)

### 1.1. Сметный раздел в составе проекта

В составе проекта (стадия "П") есть несколько разделов:
- АР — архитектурные решения
- КР — конструктивные решения
- ОВ — отопление и вентиляция
- ВК — водоснабжение и канализация
- ЭО — электрооборудование
- СС — слаботочные системы
- ПОС — проект организации строительства
- **СД** — **сметная документация** (наш раздел)

Сметный раздел опирается на **все остальные** — на чертежи, спецификации, описания технологий.

### 1.2. Состав сметной документации (стадия "П")

```
СОСТАВ КОМПЛЕКТА:
├── Локальные сметные расчёты (ЛСР) — по разделам/конструктивам
│   ├── ЛСР-01 — Демонтажные работы
│   ├── ЛСР-02 — Земляные работы
│   ├── ЛСР-03 — Конструкции бетонные
│   ├── ЛСР-04 — Кровля
│   ├── ...
│   └── ЛСР-N
├── Объектные сметы (ОС) — свод ЛСР по каждому объекту
│   ├── ОС-01 — Здание школы
│   └── ОС-02 — Благоустройство территории
├── Сводный сметный расчёт (ССР) — на стройку в целом
├── Ведомости объёмов работ (ВОР) — приложения к ЛСР
├── Ведомости ресурсов (ВР) — приложения к ЛСР или единая по комплекту
├── Расчёт ПИР (отдельно)
├── Расчёт авторского надзора (отдельно)
├── Пояснительная записка к сметной документации
└── Сводная таблица стоимости (для ТЭО, при необходимости)
```

### 1.3. Иерархия документов

```
       ССР (общий)
      ╱      ╲
   ОС-1     ОС-2
   ╱ │ ╲    ╱ ╲
ЛСР  ЛСР  ЛСР ЛСР
 │    │    │   │
 ВОР  ВОР  ВОР ВОР
 │    │    │   │
 ВР   ВР   ВР  ВР
```

Каждая ЛСР имеет свою ВОР (обоснование объёмов) и ВР (перечень ресурсов).
Каждая ОС объединяет несколько ЛСР.
ССР объединяет все ОС + лимитированные затраты.

### 1.4. Стадии проектирования и состав

#### Стадия "П" (Проект)
Полный комплект как описано выше. **Идёт в госэкспертизу**.

#### Стадия "Р" (Рабочая документация)
Уточнение/детализация. На стадии "Р" сметы могут пересчитываться, но **обычно** не меняются принципиально. Если меняются — нужна повторная экспертиза.

#### Стадия "РП" (Одностадийное)
Для простых объектов — П и Р объединены. Сметы делаются один раз.

### 1.5. Кто подписывает

| Документ | Кто подписывает |
|---|---|
| Каждая ЛСР | Сметчик-исполнитель |
| Каждая ОС | Сметчик-исполнитель + ГИП |
| ССР | Сметчик-исполнитель + ГИП + директор проектной организации |
| ВОР | Сметчик-исполнитель |
| ВР | Сметчик-исполнитель |
| Пояснительная записка | Сметчик-исполнитель + ГИП |

Подписание идёт **снизу вверх**: сначала исполнитель, потом проверяющий, потом утверждающий.

### 1.6. Передача в смежные разделы и в экспертизу

После утверждения внутри проектной организации:
1. Сметная документация **сшивается** с проектом (как раздел СД)
2. Проект **в полном составе** уходит в госэкспертизу
3. Эксперт может направить вопросы по сметному разделу
4. Сметчик готовит ответы и при необходимости вносит правки
5. После положительного заключения экспертизы — проект передаётся заказчику

### 1.7. Чек-лист комплектности

Перед сшивкой и отправкой в экспертизу — пройти по чек-листу:

- [ ] Все необходимые ЛСР присутствуют (по разделам АР, КР, ОВ, ВК, ЭО, СС, ПОС)
- [ ] Каждая ЛСР имеет свою ВОР
- [ ] Каждая ЛСР подписана
- [ ] Все ОС покрывают все объекты стройки
- [ ] ССР включает все ОС + лимитированные
- [ ] ССР по 12 главам (см. урок 2)
- [ ] Резерв на непредвиденные применён
- [ ] НДС применён
- [ ] ПИР рассчитаны отдельно
- [ ] Авторский надзор отражён
- [ ] Пояснительная записка отвечает на ключевые вопросы (см. урок 7)
- [ ] Применённая редакция СНБ — последняя
- [ ] Индексы — последний доступный квартал
- [ ] Все коэффициенты обоснованы документально
- [ ] Все индивидуальные цены имеют ≥ 3 КП

## Практика (90 мин)

### Задание

Дано: проектная документация условного объекта (учебная) — комплект чертежей АР, КР, ОВ, ВК, ЭО, ПОС `[ФАЙЛЫ: lesson-01-project-package.zip]`.

**Шаг 1:** прочитать комплект чертежей (бегло, час). Определить:
- Какие виды работ нужны
- Сколько объектов в составе стройки
- Какие специальные условия (стеснённость, действующее предприятие и т.п.)

**Шаг 2:** составить **план комплекта сметной документации**:
- Сколько ЛСР будет в сумме (типа 8-10)
- Какие именно (с названиями)
- Сколько ОС
- Что попадёт в какую главу ССР

**Шаг 3:** составить **чек-лист комплектности** для ВАШЕГО комплекта (адаптируя общий из урока).

**Шаг 4:** сделать таблицу подписей: для каждого документа — кто подпишет (имя/роль).

### Контроль

Преподаватель проверяет:
- Все нужные виды работ покрыты ЛСР (нет пропусков)
- Деление на ЛСР логично (не слишком мелко, не слишком крупно)
- Чек-лист охватывает всё
- Подписи правильно расставлены

## Самопроверка

1. Какие документы входят в сметный раздел проекта?
2. Чем ЛСР отличается от ОС, а ОС — от ССР?
3. Что такое ВОР и зачем она нужна?
4. Когда сметы делаются на стадии "П", когда на "Р", когда на "РП"?
5. Кто подписывает ЛСР? ССР?
6. Что должно быть в чек-листе перед отправкой в госэкспертизу?

## Домашнее задание

По вашему домашнему / рабочему объекту (или придумать) — составить **план комплекта** сметной документации: список ЛСР, ОС, ССР с разнесением по главам.

## Ответы

1. ЛСР, ОС, ССР, ВОР, ВР, расчёт ПИР, расчёт АН, пояснительная записка
2. ЛСР — на отдельный вид работ, ОС — свод ЛСР по объекту, ССР — на стройку в целом
3. Ведомость объёмов работ — обоснование каждого объёма формулой со ссылкой на чертёж
4. "П" — главный комплект для экспертизы; "Р" — детализация; "РП" — одностадийное для простых
5. ЛСР: сметчик-исполнитель. ССР: сметчик + ГИП + директор организации
6. Все ЛСР, все ВОР, ОС, ССР по 12 главам, резерв, НДС, ПИР, АН, пояснительная — всё обосновано
