# Урок 7. Пояснительная записка к смете

**Длительность:** 3 ак. ч (1.5 теории + 1.5 практики)

## Цель

После урока обучающийся:
- Понимает роль пояснительной записки в сметной документации
- Пишет грамотную ПЗ для проекта на госэкспертизу
- Использует ПЗ как инструмент опережающего ответа на вопросы эксперта

## Теория (70 мин)

### 7.1. Что такое пояснительная записка

**Пояснительная записка к смете** (ПЗ) — текстовый документ, объясняющий принятые сметные решения, ссылающийся на нормативную базу и обосновывающий нестандартные применения.

ПЗ — **обязательная** часть сметного раздела проекта. Без неё комплект **не принимается** в экспертизу.

### 7.2. Зачем нужна

С точки зрения эксперта:
- Эксперт читает ПЗ **первой** перед погружением в цифры
- ПЗ задаёт **контекст** для всей сметы
- ПЗ должна **отвечать на вопросы**, которые иначе эксперт задаст в замечаниях

С точки зрения проектировщика:
- ПЗ — **инструмент защиты** своей сметы
- Хорошо написанная ПЗ **сокращает количество замечаний**
- ПЗ **фиксирует** логику решений на момент сдачи (защита от "переосмыслений" со стороны заказчика через год)

### 7.3. Структура пояснительной записки

Стандартная структура (адаптировать под объект):

```
1. ОБЩИЕ ПОЛОЖЕНИЯ
   1.1. Объект и стадия проектирования
   1.2. Заказчик / проектная организация
   1.3. Нормативные документы РК, использованные при разработке
   1.4. Базисный уровень цен и квартал индексов

2. ОСНОВАНИЯ ДЛЯ РАЗРАБОТКИ
   2.1. Задание на проектирование
   2.2. Использованные исходные данные (ПОС, ТЭО, чертежи)
   2.3. Особенности объекта

3. МЕТОДЫ РАСЧЁТА
   3.1. Базисно-индексный (преимущественно)
   3.2. Индивидуальные цены — обоснование

4. НАКЛАДНЫЕ РАСХОДЫ И СМЕТНАЯ ПРИБЫЛЬ
   4.1. Применённые нормативы (по СН РК 8.02-07)
   4.2. Понижающие коэффициенты — обоснование

5. ИНДЕКСЫ ПЕРЕСЧЁТА
   5.1. Источник, квартал
   5.2. Способ применения

6. КОЭФФИЦИЕНТЫ УСЛОВИЙ ПРОИЗВОДСТВА
   6.1. Перечень применённых коэффициентов
   6.2. Обоснование каждого с ссылкой на ПОС / приказ / акт обследования

7. ИНДИВИДУАЛЬНЫЕ ЦЕНЫ МАТЕРИАЛОВ
   7.1. Перечень материалов с индивидуальными ценами
   7.2. По каждому — основание (3 КП), выбор, обоснование

8. ЛИМИТИРОВАННЫЕ ЗАТРАТЫ
   8.1. Временные здания (расчёт по нормативу)
   8.2. Зимнее удорожание (привязка к ПОС)
   8.3. Непредвиденные (норматив для типа объекта)
   8.4. Содержание заказчика (по СН РК 8.02-05)
   8.5. Авторский надзор

9. ПРОЕКТНО-ИЗЫСКАТЕЛЬСКИЕ РАБОТЫ
   9.1. Состав
   9.2. Способ расчёта (норматив / СБЦ / калькуляция)

10. ИТОГОВАЯ СТОИМОСТЬ
    10.1. Структура (СМР / ПИР / лимитированные / резерв / НДС)
    10.2. Сравнение с аналогичными объектами (если данные есть)

11. СПЕЦИАЛЬНЫЕ ПРИМЕЧАНИЯ
    11.1. Решения, которые могут вызвать вопросы (упреждающие ответы)
```

### 7.4. Принципы хорошей ПЗ

#### Принцип 1: Краткость + полнота
Не "много текста", а "достаточно, чтобы читатель не задавал вопросов". Обычно 5-15 страниц для среднего объекта.

#### Принцип 2: Прямые ссылки
Каждое утверждение — со ссылкой на пункт нормативки или на конкретный документ:
- "Применён коэффициент 1.15 за работу на действующем объекте по СН РК 8.02-XX п. X.X.X на основании приказа директора школы №47 от 15.04.2026 о режиме ремонтных работ в каникулярный период"

#### Принцип 3: Опережающие ответы
Если вы знаете, что какое-то решение может показаться странным эксперту — **сразу объясните**. Иначе придёт замечание, и вы потратите время на ответ.

#### Принцип 4: Без воды
Никаких "общих фраз". Только конкретика. "Сметы разработаны в соответствии с нормативной базой" — пустая фраза. "Сметы разработаны на основе СН РК 8.02-01 (ред. 2024) с применением СНБ РК на 01.01.2024 и индексов 2 квартала 2026" — конкретика.

#### Принцип 5: Чёткая структура
Читатель должен **за 30 секунд** найти раздел с ответом на свой вопрос. Поэтому:
- Заголовки и подзаголовки
- Выделение ключевых цифр
- Таблицы вместо длинных абзацев

### 7.5. Типовые ошибки в ПЗ

| Ошибка | Что не так |
|---|---|
| Отсутствие ПЗ | Самая грубая — комплект не принимается |
| ПЗ "ни о чём" | Общие фразы без конкретики, не отвечает на вопросы |
| Устаревшие ссылки | Ссылка на старую редакцию СН РК |
| Отсутствие ссылок на ПОС | Коэффициенты без обоснования через ПОС |
| Без обоснования индивидуальных цен | "Цена принята согласно КП" — мало; нужно: 3 КП, выбор, основание |
| Не объяснены допущения | Использовали нестандартный коэффициент — не сказали почему |
| Длинная без структуры | Эксперт не находит ответ на свой вопрос → задаёт замечание |

## Практика (90 мин)

### Задание

Дано: уже составленный сметный комплект учебного объекта (вы делали в предыдущих уроках).

**Шаг 1:** написать **полную пояснительную записку** к этому комплекту по структуре из раздела 7.3. Объём — 8-12 страниц.

Главное:
- Каждый коэффициент — с ссылкой на пункт ПОС / приказ / акт
- Каждая индивидуальная цена — с обоснованием
- Все нормативы НР, СП, индексов — с источниками
- Раздел "Специальные примечания" — упреждающие ответы

**Шаг 2:** **Кросс-проверка** с однокурсником: дать друг другу почитать. Проверить:
- Понятна ли логика?
- Можно ли найти ответ на любой "если эксперт спросит..."?
- Все ли странные места объяснены?

**Шаг 3:** **самокритика** — пройти по списку типовых ошибок (раздел 7.5) и убедиться, что вашей ПЗ нет ни одной.

### Контроль

Куратор оценивает:
- Структура соблюдена
- Все обоснования на месте
- Нет общих фраз без конкретики
- Каждое нестандартное решение объяснено

## Домашнее задание

Написать ПЗ к вашему учебному / рабочему объекту, прошедшему через все предыдущие уроки.

## Ответы

1. Текстовый документ, объясняющий принятые сметные решения и ссылающийся на нормативку
2. Да — без неё комплект не принимается в экспертизу
3. Общие положения, основания, методы расчёта, НР и СП, индексы, коэффициенты, индивидуальные цены, лимитированные, ПИР, итоговая, спецпримечания (любые 5)
4. 5-15 страниц для среднего объекта
5. Заранее объяснять решения, которые могут показаться эксперту странными — чтобы не получить замечание
6. Отсутствие ПЗ; общие фразы; устаревшие ссылки; нет обоснования коэффициентов / индивидуальных цен (любые 3)
7. Ссылка на пункт нормативки + конкретный документ-подтверждение (приказ, акт, ПОС)
