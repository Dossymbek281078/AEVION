# Урок 3. Терминология сметного дела

**Длительность:** 4 ак. ч (2 теории + 2 практики)

## Цель

После урока обучающийся:
- Знает 30 ключевых терминов сметного дела РК
- Может объяснить каждый термин одним предложением
- Различает близкие по смыслу термины (ЛСР vs ОС, НР vs СП, ПЗ vs полная стоимость)

## Теория (90 мин)

### 3.1. Документы

| Термин | Расшифровка | Что это |
|---|---|---|
| **ЛСР** | Локальный сметный расчёт | Смета на отдельный вид работ или раздел в составе одного объекта |
| **ОС** | Объектная смета | Свод ЛСР по одному объекту строительства |
| **ССР** | Сводный сметный расчёт | Общий расчёт по стройке: сумма всех ОС + лимитированные затраты + резерв + НДС |
| **ВОР** | Ведомость объёмов работ | Таблица обоснования объёмов: формула расчёта со ссылкой на чертёж |
| **ВР** | Ведомость ресурсов | Перечень всех ресурсов сметы с количествами и ценами |
| **ПЗ** | Пояснительная записка | Текстовое обоснование принятых сметных решений |

### 3.2. Стадии и виды работ

| Термин | Что это |
|---|---|
| **ПИР** | Проектно-изыскательские работы (проектирование + инженерные изыскания) |
| **СМР** | Строительно-монтажные работы (само строительство) |
| **АН** | Авторский надзор (контроль проектировщика на стройке) |
| **Стадия "П"** | Стадия "Проект" — основной комплект документации |
| **Стадия "Р"** | Стадия "Рабочая документация" — детальные чертежи для производства |
| **Стадия "РП"** | Одностадийное проектирование (для простых объектов) |

### 3.3. Виды строительства

| Термин | Что это |
|---|---|
| **Новое строительство** | Возведение объекта на новом месте |
| **Реконструкция** | Изменение параметров существующего объекта (площадь, этажность, назначение) |
| **Капитальный ремонт** | Восстановление эксплуатационных характеристик без изменения параметров |
| **Текущий ремонт** | Регулярные мелкие работы для поддержания эксплуатации |
| **Реставрация** | Восстановление объектов культурного наследия |
| **Демонтаж / снос** | Разборка объекта целиком |

### 3.4. Стоимостные понятия

| Термин | Что это |
|---|---|
| **Прямые затраты (ПЗ)** | Стоимость трудозатрат + ЭМ + материалов по позиции |
| **Накладные расходы (НР)** | Затраты на содержание организации, не привязанные к конкретной работе |
| **Сметная прибыль (СП)** | Нормируемая прибыль подрядчика |
| **ФОТ** | Фонд оплаты труда (трудозатраты рабочих + зарплата машинистов в ЭМ) |
| **ЭМ** | Эксплуатация машин (стоимость машино-часа + зарплата машиниста) |
| **Лимитированные затраты** | Затраты, добавляемые к итогу (временные здания, зимнее удорожание, непредвиденные) |
| **Резерв на непредвиденные** | Норматив запаса на случай изменений |

### 3.5. Цены и индексы

| Термин | Что это |
|---|---|
| **Базисный уровень цен** | Год и регион, на которые зафиксированы цены в нормативной базе |
| **Текущие цены** | Цены на момент составления / актирования сметы |
| **Индекс пересчёта** | Коэффициент перевода базисных цен в текущие |
| **Базисно-индексный метод** | Расчёт в базисных ценах × индекс |
| **Ресурсный метод** | Расчёт сразу в текущих ценах ресурсов |
| **Индивидуальная цена** | Цена ресурса по фактическим коммерческим предложениям |

### 3.6. Нормативная база

| Термин | Что это |
|---|---|
| **СНБ** | Сметно-нормативная база |
| **СН РК** | Строительные нормы Республики Казахстан |
| **ГЭСН РК** | Государственные элементные сметные нормы |
| **ЕРЕР РК** | Единичные расценки на ремонтно-строительные работы (или общестроительные) |
| **Сборник цен** | Справочник цен на материалы / эксплуатацию машин / тарифные ставки |
| **КДС** | Комитет по делам строительства МИИР РК |

### 3.7. Исполнительная документация

| Термин | Что это |
|---|---|
| **КС-2** | Акт о приёмке выполненных работ |
| **КС-3** | Справка о стоимости выполненных работ и затрат |
| **КС-6а** | Журнал учёта выполненных работ |
| **Дефектный акт** | Документ о выявленных недостатках или дополнительных работах |
| **Допсоглашение** | Дополнение к договору при изменении объёмов или цены |

## Практика (90 мин)

### Часть 1. Кроссворд (45 мин)

Преподаватель раздаёт кроссворд из 30 терминов с подсказками. Обучающиеся заполняют на время. Сверяют ответы.

### Часть 2. Карточки (45 мин)

50 карточек: на одной стороне термин, на другой — определение. Работа в парах:
- Один показывает термин — другой даёт определение
- Меняются ролями
- За 30 минут пройти весь стек 2 раза

Контроль: преподаватель спрашивает по случайной выборке из 10 терминов.

## Самопроверка

1. Чем ЛСР отличается от ОС?
2. Что такое ССР и из чего он состоит?
3. Чем НР отличаются от СП?
4. Что входит в "прямые затраты" по позиции?
5. Чем стадия "П" отличается от стадии "Р"?
6. Чем реконструкция отличается от капитального ремонта?
7. Что такое базисно-индексный метод?
8. Что такое лимитированные затраты, приведите 3 примера?
9. Куда относится зарплата машиниста — в ФОТ или в стоимость материалов?
10. Что такое КС-2 и КС-3?

## Домашнее задание

Составить **личный глоссарий** на основе урока: ваше определение каждого из 30 терминов + одно собственное предложение-пример. Сдать на проверку.

## Ответы

1. ЛСР — на отдельный вид работ; ОС — на объект целиком (свод ЛСР)
2. Сводный сметный расчёт. Состав: главы 1-12 (объекты + лимитированные) + резерв + НДС
3. НР — содержание организации; СП — нормируемая прибыль
4. Трудозатраты + ЭМ + материалы
5. "П" — основной проект; "Р" — детальные рабочие чертежи
6. Реконструкция меняет параметры объекта; капремонт — восстанавливает без изменения параметров
7. Расчёт в базисных ценах × индекс пересчёта на текущий период
8. Затраты сверх ПЗ+НР+СП. Примеры: временные здания, зимнее удорожание, непредвиденные
9. В ФОТ (через ЭМ) — это часть фонда оплаты труда, влияющая на расчёт НР и СП
10. КС-2 — акт о приёмке выполненных работ; КС-3 — справка о стоимости (нарастающим итогом)
