# raw-corpus — нормативные источники

Сюда складываются исходные нормативные документы РК и скрипты их парсинга.
Сами тяжёлые исходники в git не коммитятся (см. `.gitignore`), коммитим
только парсеры. Готовые JSON копируются в `frontend/public/ssc/` (для ССЦ).

## ССЦ РК 8.04-08-2025 — Сметные цены на материалы

**Источник:** `C:\Users\user\Downloads\ССЦ РК 8.04-08-2025 Сметные цены на строительные материалы.rar`

**Шаги воспроизведения:**

```powershell
# 1. Распаковать архив
mkdir raw-corpus\ssc-rk-2025
& "C:\Program Files\WinRAR\UnRAR.exe" x -o+ `
    "C:\Users\user\Downloads\ССЦ РК 8.04-08-2025 Сметные цены на строительные материалы.rar" `
    "C:\Users\user\aevion-smeta-trainer\raw-corpus\ssc-rk-2025\"

# 2. Распарсить (Python 3, stdlib only)
python -X utf8 raw-corpus\parse-ssc.py

# 3. Скопировать в публичную статику фронта
Copy-Item raw-corpus\parsed\*.json frontend\public\ssc\ -Force
```

**Что получается:**
- 92 JSON-файла (по сборнику)
- ~33 МБ суммарно
- 182 936 материалов
- Покрытие: 17 регионов × Кн.1 (основные материалы) + Кн.7 (бетон/ж/б) +
  общие Кн.2-6 + Общие положения

**Структура одной записи:**
```json
{ "code": "211-101-0102", "name": "Земля растительная",
  "unit": "м3", "cargoClass": "1", "grossKg": 1200,
  "smetnaya": 3154, "otpusknaya": 1316 }
```
- `smetnaya` — сметная цена (с ТЗР), применяется в смете
- `otpusknaya` — отпускная цена производителя
- `isGroup: true` — это заголовок группы (без цены), а не материал
