# AEVION handoff (dev)

Эта папка нужна, чтобы быстро продолжить разработку без потери контекста.

Что внутри:
- Краткие статусы модулей (`qright_module_status.md` и далее по мере разработки)
- Шаблоны промптов для Cursor-чата по каждому приложению
- Шаблон “склейки” (интеграция модулей в общий проект)

Как пользоваться:
1) Создай новый Chat в Cursor для нужного модуля.
2) Вставь соответствующий шаблон из папки `prompts/`.
3) Продолжай работу строго по проверкам “backend route / frontend page / persist / test”.
