# P3-1 — Investor Demo Script (8 минут)

> Скрипт под запись unlisted-видео для инвесторов. Цель: за 8 минут показать
> полный цикл AEVION одним сквозным кейсом.
>
> **Подготовка перед записью:**
> 1. Прогнать `bash scripts/investor-demo-audit.sh` → должно быть 18/18 green.
> 2. Залогиниться заранее на `aevion.app` под тестовой учёткой.
> 3. Подготовить файл-«произведение» — небольшая JPG картинка или MP3-семпл (5-10 МБ).
> 4. Открыть Stripe тест-карту в notepad: `4242 4242 4242 4242`, любой CVC, exp 12/30.
> 5. OBS / Loom: запись 1920×1080, 30fps, mic gain нормализован.
> 6. Браузер: Chrome incognito + закладочная панель скрыта.
>
> **Кейс:** автор регистрирует музыкальный трек, получает Bureau cert,
> попадает в Trust Graph, забирает AEC награду.

## 0:30 — 1:30  QRight — регистрация

**Экран:** клик в навигации `QRight` → `aevion.app/qright`.

**Голос:**
> «Шаг первый — QRight. Я загружаю аудиофайл — это мой трек. Платформа
> моментально считает SHA-256 хеш содержания. Этот хеш — отпечаток
> произведения, любая правка ломает его. Дальше — anchor в Bitcoin
> через OpenTimestamps: время фиксируется на blockchain'е, который
> я не контролирую. Это технический эквивалент ‘у меня было раньше’.»

**Действия:**
1. Drag-and-drop аудиофайл в зону загрузки.
2. Дождаться появления SHA-256 (1-2 сек) — крупный курсор на хеш.
3. Клик «Зарегистрировать».
4. Показать получившуюся карточку записи: hash + timestamp + статус.

## 1:30 — 3:00  QSign — постквантовая подпись

**Экран:** навигация → `QSign` → `aevion.app/qsign`.

**Голос:**
> «QSign подписывает зарегистрированное произведение. Не RSA, не ECDSA.
> Мы используем ML-DSA-65 — это стандарт NIST FIPS 204, **постквантовая
> криптография**. Когда квантовые компьютеры сломают RSA, наши подписи
> останутся валидными. На рынке B2C это есть только у нас.»
>
> «В дополнение — Quantum Shield: ключ автора рассыпается на три части
> по схеме Shamir 2-of-3. Любые две из трёх восстанавливают подпись.
> Это значит: если устройство автора утеряно, доступ не теряется.»

**Действия:**
1. Выбрать запись из QRight (dropdown).
2. Клик «Подписать ML-DSA-65».
3. Показать сгенерированную подпись + публичный ключ.
4. Перейти на `/verify` — вставить hash → показать «✓ valid».
