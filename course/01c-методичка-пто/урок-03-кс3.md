# Урок 3. Справка КС-3

**Длительность:** 3 ак. ч (1 теория + 2 практики)

## Цель

После урока обучающийся:
- Понимает разницу между КС-2 и КС-3
- Корректно формирует КС-3 нарастающим итогом
- Видит связь КС-3 с договором и оплатами

## Теория (45 мин)

### 3.1. Что такое КС-3

**КС-3** — "Справка о стоимости выполненных работ и затрат" — обобщающий документ, показывающий **общую стоимость работ** на объекте за отчётный период и **нарастающим итогом с начала строительства**.

**Связь с КС-2:**
- КС-2 — детальный акт по позициям за конкретный период
- КС-3 — суммарный итог по всем КС-2, с разбивкой на "за период" и "с начала"

### 3.2. Структура КС-3

```
┌──────────────────────────────────────────────┐
│ ШАПКА: реквизиты, объект, № справки, период  │
└──────────────────────────────────────────────┘
┌──────────────────────────────────────────────┐
│ ТАБЛИЦА                                       │
│ Наименование │ С нач. стр-ва │ С нач. года │ За пер.│
│ ──────────────┼───────────────┼────────────┼────────│
│ СМР по ЛСР   │     ...       │    ...     │  ...   │
│ Допработы    │     ...       │    ...     │  ...   │
│ Зимнее удор. │     ...       │    ...     │  ...   │
│ ИТОГО        │     ...       │    ...     │  ...   │
│ В т.ч. НДС   │     ...       │    ...     │  ...   │
└──────────────────────────────────────────────┘
┌──────────────────────────────────────────────┐
│ ПОДПИСИ                                       │
└──────────────────────────────────────────────┘
```

### 3.3. Логика "с начала строительства"

```
Месяц 1: КС-2 на 5 млн → КС-3 (с начала: 5 млн, за период: 5 млн)
Месяц 2: КС-2 на 7 млн → КС-3 (с начала: 12 млн, за период: 7 млн)
Месяц 3: КС-2 на 4 млн → КС-3 (с начала: 16 млн, за период: 4 млн)
...
Итоговый КС-3 после всех КС-2 = ИТОГ ПО ОБЪЕКТУ.
```

Графа **"С начала года"** имеет значение для бухгалтерии заказчика (особенно бюджетного — годовой бюджет сбрасывается).

### 3.4. Зачем нужен КС-3 отдельно от КС-2

- Бухгалтерия работает с **обобщёнными** документами, КС-2 им детализирован избыточно
- Налоговая — хочет видеть итог за период
- Заказчик — может не интересоваться позициями, только суммой
- При большом числе КС-2 за период (например, по нескольким разделам) — КС-3 их сводит

### 3.5. Связь с оплатами

```
КС-3 подписан → бухгалтерия заказчика → платёжное поручение → деньги подрядчику
```

КС-3 — **основание для оплаты**. Без КС-3 заказчик платить не должен.

В договоре прописывается срок оплаты после подписания КС-3 (например, "в течение 10 рабочих дней с момента подписания справки КС-3 заказчиком").

### 3.6. Когда разделять и когда нет

#### КС-2 + КС-3 на каждый период
Стандартная практика. Каждый месяц — пара "КС-2 + КС-3".

#### Несколько КС-2, один КС-3 за период
Когда работы делятся по разделам/субподрядчикам, и каждый сдаёт свой КС-2, а КС-3 формируется общий.

#### Без КС-3 при коротком объекте
На очень мелких объектах (1 акт за всё) — КС-3 может не выпускаться, итог проводится через КС-2 + договор.

## Демонстрация (15 мин)

Преподаватель:
1. Открывает уже сделанный КС-2 за июнь и за июль
2. Показывает, как программа автоматически формирует КС-3 за июль (с разбивкой "за период" / "с начала")
3. Сравнивает с тем, что было бы, если бы КС-3 формировался только за июль (без нарастающего)

## Параллельная практика (90 мин)

### Задание

Дано: серия из 4 готовых КС-2 за май, июнь, июль, август `[ФАЙЛЫ: lesson-03-ks2-MM.pdf]`.

**Шаг 1:** в каждой программе сформировать КС-3 за **август** (нарастающим итогом).

**Шаг 2:** сверить графы:
- Стоимость с начала строительства = сумма всех 4 КС-2
- За период (август) = только КС-2 за август

**Шаг 3:** проверить арифметику вручную в Excel. Расхождение между программой и ручным расчётом не должно превышать копейки.

**Шаг 4:** усложнение — допустим, в июле был дополнительный КС-2 на допработы (5-й документ). Включить его в КС-3 (отдельной строкой "Допработы") и пересчитать.

### Контроль

Куратор проверяет: КС-3 правильно нарастает по всем КС-2, допработы выведены отдельно, итоги сходятся.

## Самопроверка

1. Чем КС-3 отличается от КС-2?
2. Что такое "нарастающим итогом"?
3. Зачем нужны три графы (с начала строительства / с начала года / за период)?
4. Кто и кому платит на основе КС-3?
5. Может ли быть КС-2 без КС-3?
6. Как в КС-3 отражаются допработы?

## Домашнее задание

По 6 КС-2 (учебных) за полугодие — сформировать **итоговый КС-3** на конец июня. Сдать в одной программе на выбор.

## Ответы

1. КС-2 — детальный акт за период; КС-3 — обобщённая справка нарастающим итогом
2. Сумма за все периоды от начала строительства до текущего включительно
3. Бухгалтерия заказчика: с начала строительства — общий контроль; с начала года — годовой бюджет; за период — текущий платёж
4. Заказчик платит подрядчику; КС-3 — основание для оплаты
5. На очень мелких объектах — может; стандартно — нет
6. Отдельной строкой в таблице (на основе допсоглашения и отдельного КС-2)
