# LMS — чек-листы для обучающихся и кураторов

Чек-листы — рабочие инструменты для самопроверки и проверки преподавателем. Применяются на каждом уровне.

## Чек-лист для обучающегося: "Перед сдачей ЛСР" (уровень 2+)

Использовать перед нажатием "Отправить на проверку".

- [ ] Заполнен титул: объект, наименование работ, ФИО, базисный уровень
- [ ] Все позиции дефектной ведомости / задания отражены в смете
- [ ] Все объёмы перепроверены: единицы измерения нормы соответствуют введённым числам
- [ ] Под каждый объём в ВОР есть формула расчёта с привязкой к чертежу
- [ ] Корректный вид работ (новое строительство / ремонт / реконструкция)
- [ ] Корректные нормативы НР и СП по виду работ
- [ ] Применённые коэффициенты условий имеют обоснование (приказ / акт / фотофиксация)
- [ ] Применены актуальные индексы пересчёта на нужный квартал
- [ ] Если есть индивидуальные цены — снят флажок применения индекса к ним
- [ ] Если есть индивидуальные цены — приложены ≥ 3 КП и обоснование выбора
- [ ] Сумма прямые + НР + СП + лимитированные + НДС = итог
- [ ] Сформированы: форма №1, ВОР, ВР, пояснительная записка
- [ ] При работе в двух программах: расхождение итога ≤ 0.5%
- [ ] Пояснительная записка отвечает на "почему" по каждому неочевидному решению

## Чек-лист куратора: проверка ЛСР (уровень 2+)

- [ ] Структура документа: ЛСР / ВОР / ВР / пояснительная записка — все на месте
- [ ] Объёмы (выборочно по 3 позициям): сходятся с чертежом
- [ ] Расценки: применены подходящие, единицы измерения корректны
- [ ] НР/СП: верный норматив, верная база начисления
- [ ] Коэффициенты: применение обосновано, ссылка на нормативку, не превышают разрешённые СН РК
- [ ] Индексы: актуальные, применены на правильный элемент
- [ ] Индивидуальные цены: 3+ КП, обоснование выбора, индекс не применился
- [ ] Двойного счёта нет (между позициями, между ЛСР и лимитированными)
- [ ] Нет позиций "по аналогии" без обоснования
- [ ] Округления корректны
- [ ] Пояснительная записка читается без подсказок: внешний эксперт поймёт логику

## Чек-лист обучающегося: "Перед сдачей КС-2" (уровень 3)

- [ ] КС-2 за тот период, который актируется (даты в шапке)
- [ ] Все позиции КС-2 опираются на договорную цену / основную ЛСР / допсоглашение
- [ ] Объёмы по КС-2 ≤ объёмов по графику и журналу работ за период
- [ ] Применённые коэффициенты на этом КС-2 имеют обоснование (приказ заказчика, фото)
- [ ] КС-2 содержит подписи: подрядчика → технадзора → заказчика (или отметка о готовности к подписанию)
- [ ] Сумма КС-2 включается в КС-3 нарастающим итогом — арифметика проверена
- [ ] Если есть допработы — отдельный КС-2 с ссылкой на допсоглашение
- [ ] Если применено зимнее удорожание — это работы при отрицательной температуре, подтверждено справкой о температуре

## Чек-лист куратора: проверка КС-2 (уровень 3)

- [ ] Соответствие КС-2 договорной цене (только то, что в договоре, или есть ДС)
- [ ] Объёмы соответствуют журналу работ
- [ ] Объёмы не выходят за объёмы основной ЛСР по этой позиции (за весь период)
- [ ] Применение коэффициентов: обосновано, документировано, корректно
- [ ] Зимнее удорожание: применено только к работам, на которые влияет температура, и только в холодные месяцы
- [ ] Допработы: пакет полный (дефектный акт, согласование, ЛСР, ДС, отдельный КС-2)
- [ ] Связь КС-2 → КС-3 нарастающим итогом — арифметика верна

## Чек-лист обучающегося: "Перед сдачей комплекта на экспертизу" (уровень 4)

- [ ] ЛСР по всем разделам проекта присутствуют (по чек-листу комплектности)
- [ ] Объектные сметы корректно сгруппированы из ЛСР
- [ ] ССР по 12 главам, каждое лимитированное в своей главе
- [ ] Резерв на непредвиденные — нормативный для данного типа объекта (бюджет / коммерция)
- [ ] НДС применён на итог
- [ ] ПИР рассчитан отдельно, авторский надзор отражён
- [ ] Пояснительная записка содержит:
  - [ ] Перечень нормативки с редакциями
  - [ ] Базисный уровень и квартал индексов
  - [ ] Обоснование каждого коэффициента с ссылкой на пункт СН РК
  - [ ] Обоснование каждой индивидуальной цены
  - [ ] Расчёт лимитированных по главам ССР
- [ ] Все ВОР приложены и читаемы
- [ ] ВР итоговая по всему комплекту присутствует
- [ ] Файл проекта с расчётами в исходнике приложен (для проверки экспертом)

## Чек-лист эксперта: проверка комплекта (уровень 5)

50 пунктов, разбитых по 9 слоям:

### Слой 1. Комплектность (5 пунктов)
- [ ] Все ЛСР присутствуют по перечню (по разделам проекта)
- [ ] Объектные сметы охватывают все объекты стройки
- [ ] ССР включает все объекты + лимитированные
- [ ] ВОР приложены ко всем ЛСР
- [ ] Пояснительная записка присутствует и подписана

### Слой 2. Нормативка (5 пунктов)
- [ ] Применённая редакция СНБ актуальна
- [ ] Индексы — последний доступный квартал
- [ ] СН РК 8.02-01 / 8.02-02 / 8.02-07 — корректные ссылки
- [ ] Коэффициенты по техническим частям сборников применены без противоречий
- [ ] Учёт особенностей объекта (новое / реконструкция / ремонт) корректен

### Слой 3. Соответствие проекту (8 пунктов)
- [ ] Каждая позиция в смете — есть в проекте
- [ ] Технологии работ соответствуют проектным решениям
- [ ] Материалы соответствуют спецификациям
- [ ] Оборудование соответствует спецификациям
- [ ] Объёмы по выборочной проверке = объёмам по чертежам
- [ ] Учтены этапность и поэтапная сдача (если применимо)
- [ ] Учтены особенности ПОС
- [ ] Учтены ограничения ТЗ

### Слой 4. Корректность объёмов (5 пунктов)
- [ ] Выборочная проверка 10 позиций — расхождение ≤ 0.5%
- [ ] Проёмы вычтены корректно
- [ ] Откосы посчитаны как отдельная позиция
- [ ] Нет дублирования объёмов между позициями
- [ ] Округления выполнены корректно

### Слой 5. Корректность расценок (6 пунктов)
- [ ] Выбраны точные расценки (нет необоснованного применения "по аналогии")
- [ ] Единицы измерения нормы и объёма согласованы
- [ ] Шифры расценок соответствуют актуальным сборникам
- [ ] Индивидуальные расценки имеют расчёт-обоснование
- [ ] Корректировки состава расценки обоснованы
- [ ] Нет применения более дорогой нормы при наличии точной

### Слой 6. Ресурсы и индивидуальные цены (5 пунктов)
- [ ] Базисные цены ресурсов — из актуального сборника
- [ ] Индивидуальные цены — ≥ 3 КП на каждую
- [ ] Выбор цены из КП обоснован
- [ ] Транспортные расходы учтены корректно (франко-склад / франко-объект)
- [ ] К индивидуальным ценам не применён индекс

### Слой 7. НР и СП (4 пункта)
- [ ] Норматив НР соответствует виду работ по СН РК 8.02-07
- [ ] Норматив СП соответствует
- [ ] База начисления выбрана корректно (ФОТ / ПЗ — по виду работ)
- [ ] Понижающие коэффициенты применены с обоснованием

### Слой 8. Лимитированные затраты (5 пунктов)
- [ ] Временные здания — не дублируются с ЛСР
- [ ] Зимнее удорожание — только на работы, на которые влияет температура, и только в холодные месяцы (по графику)
- [ ] Непредвиденные — норматив для типа объекта
- [ ] Содержание заказчика — по СН РК 8.02-05
- [ ] Авторский надзор — по нормативу

### Слой 9. Арифметика и формы (7 пунктов)
- [ ] Итоги по ЛСР сходятся с программным расчётом
- [ ] Итоги ОС = сумма ЛСР + лимитированные ОС
- [ ] Итоги ССР = сумма ОС + лимитированные ССР по главам
- [ ] Резерв непредвиденных применён к нужной базе
- [ ] НДС применён к итогу
- [ ] Округления на каждом уровне согласованы
- [ ] Все формы (1, 2, 3, ВОР, ВР) выгружены и читаемы

## Чек-лист защиты перед комиссией (уровень 5)

Этот чек-лист — для самой комиссии при оценке защиты.

- [ ] Обучающийся представил структуру своего заключения
- [ ] Каждое замечание привязано к конкретному пункту нормативки
- [ ] При оппонировании удерживает позицию, где прав
- [ ] При оппонировании принимает позицию, где не прав
- [ ] Аргументирует через нормативку, а не через "мне кажется"
- [ ] Корректен в формулировках (не оскорбителен к "автору")
- [ ] Знает финансовый эффект своих замечаний (в тенге)
- [ ] Может предложить альтернативное решение (не только "это плохо")
