## AEVION выходит из закрытой беты: единый pipeline для авторов в эпоху ИИ-контента

**Алматы, Казахстан** — AEVION, платформа для цифровой верификации и монетизации интеллектуальной собственности, объявляет об открытом запуске на [дата].

### Что произошло

После 30 дней разработки (130+ merged PR, 600+ коммитов) платформа достигла production-ready состояния с 16 работающими модулями. С сегодняшнего дня платформа доступна на **aevion.app** без ограничений.

### Проблема

ИИ ежедневно генерирует миллиарды единиц контента. Граница между оригинальным творчеством и копией исчезла. Чтобы доказать авторство, проверить происхождение работы и получить роялти, создатель сегодня жонглирует четырьмя разорванными платформами: регистрацией, подписью, мониторингом и выплатами.

### Решение

AEVION собирает это в один pipeline: **register → sign → certify → vote → earn**.

Каждый шаг криптографически связан с предыдущим:
- **QRight** фиксирует SHA-256 хеш с временной меткой
- **QSign** добавляет постквантовую подпись ML-DSA-65 (NIST FIPS 204)
- **Bureau** выдаёт юридически значимый сертификат через Trust Graph
- **Planet** запускает compliance-валидацию с открытым Merkle-snapshot
- **Bank** автоматически начисляет AEC-роялти при каждой верификации

### Технические особенности

**Постквантовая криптография.** Алгоритм ML-DSA-65 (NIST FIPS 204, финализирован август 2024) защищает подписи от будущих квантовых атак. Это решает проблему "harvest-now, decrypt-later": государства и корпорации уже сегодня сохраняют зашифрованный трафик для расшифровки через 7–10 лет. Подписи AEVION останутся юридически значимыми.

**Trust Graph.** Каждое действие пользователя — создание, верификация, голосование — добавляет ребро в граф доверия. Граф — это долгосрочный актив: конкурент, который начнёт с нуля сегодня, накопит эквивалентную репутацию через 2–3 года.

**Производительность и надёжность.** 24-шаговый prod smoke запускается ежедневно через GitHub Actions. 130+ PR прошли code review и CI перед мерджем.
