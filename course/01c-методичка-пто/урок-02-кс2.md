# Урок 2. Акт КС-2

**Длительность:** 6 ак. ч (2 теории + 4 практики)

## Цель

После урока обучающийся:
- Знает, что такое КС-2 и зачем он нужен
- Составляет КС-2 в каждой из трёх программ
- Знает порядок подписания и типичные ошибки

## Теория (90 мин)

### 2.1. Что такое КС-2

**КС-2** — "Акт о приёмке выполненных работ" — первичный учётный документ, фиксирующий, какие работы и в каком объёме фактически выполнены подрядчиком за отчётный период.

**Зачем:**
- Подрядчик подтверждает выполнение
- Заказчик принимает (или не принимает с замечаниями)
- На основе КС-2 формируется КС-3 (справка о стоимости) и идёт оплата
- КС-2 — основа бухгалтерского учёта на стройке для всех сторон

### 2.2. Когда составляется

Чаще всего:
- **Помесячно** — стандартный режим
- **Поэтапно** — при поэтапной приёмке (например, после завершения каждого крыла здания)
- **По факту приёмки** — для отдельных видов работ (например, фундамент сдан)

Регулярность фиксируется в договоре подряда.

### 2.3. Структура КС-2

```

│ ...                                           │

┌──────────────────────────────────────────────┐
│ НАЧИСЛЕНИЯ                                    │
│ - НР, СП                                      │
│ - Коэффициенты условий                        │

└──────────────────────────────────────────────┘
```

### 2.4. Связь с ЛСР и договором

КС-2 **должен** опираться на:
- ЛСР, приложенную к договору (та же позиция, тот же шифр расценки, та же цена)
- Фактически выполненные объёмы (по журналу работ КС-6а)

КС-2 **не должен** содержать:
- Позиции, которых нет в ЛСР (если это допработы — отдельный КС-2 на основе допсоглашения)
- Объёмы больше, чем по ЛСР (за весь срок производства; в одном акте может быть текущий период)
- Цены, отличающиеся от ЛСР (без оснований)

### 2.5. Порядок подписания

```
1. ПТО подрядчика → составляет КС-2 в программе
2. Производитель работ / прораб → проверяет соответствие журналу
3. Подрядчик (директор/уполномоченное лицо) → подписывает "Сдал"
4. Технический надзор → проверяет работы по факту, подписывает или возвращает с замечаниями
5. Заказчик → подписывает "Принял"
6. Бухгалтерия заказчика → принимает к учёту
```

Если **технадзор не подписывает** — указывает замечания письменно. Подрядчик устраняет, заново сдаёт. Это нормально, не конфликт.

Если **заказчик не подписывает** при подписанном технадзоре — требует обоснований. Или это юридический спор.

### 2.6. Типичные ошибки в КС-2

| Ошибка | Описание | Последствия |
|---|---|---|
| Объём больше, чем сделано | По журналу работ выполнено меньше, чем заявлено в КС-2 | Технадзор отказывает в подписании |
| Не та позиция | В КС-2 расценка, отличная от ЛСР | Снимается при проверке, акт переподписывается |
| Применён коэффициент без основания | Стеснённые условия, действующее предприятие — без подтверждения | Снимается, объясниться с экспертом |
| Зимнее удорожание не на тех работах | Применено к работам, не чувствительным к температуре | Снимается |
| Допработы вписаны без допсоглашения | "Заодно сделали — оплатите" | Заказчик имеет право не платить |
| Нумерация и даты с ошибками | Период перекрывается с другим актом | Требует пересоставления |

### 2.7. КС-2 в трёх программах

Все программы поддерживают формирование КС-2 **на основе уже существующей ЛСР**:

- **АВС-4**: меню "Документ → Создать акт КС-2 на основе сметы" → выбрать позиции и проставить выполненные объёмы.
- **Смета РК**: вкладка "Актирование", создание акта на основе сметы, ввод объёмов.
- **Сана**: модуль "Акты", выбор сметы, ввод фактических объёмов в столбец "Факт".

Поведение одинаковое: программа берёт цены из ЛСР, умножает на введённые фактические объёмы, считает НР/СП/коэффициенты, выдаёт акт.

6. Формирует печатную форму КС-2 (PDF)

## Ответы

1. Акт о приёмке выполненных работ — фиксирует, что фактически выполнено за период
2. Помесячно (стандарт), поэтапно или по факту приёмки
3. На ЛСР, приложенную к договору, и на фактически выполненные объёмы (журнал работ)
4. Подрядчик → технадзор → заказчик
5. Получить замечания, устранить, переподписать
6. Заказчик имеет право отказать в оплате этой части до оформления допсоглашения
