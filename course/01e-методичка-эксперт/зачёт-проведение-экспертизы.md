# Зачёт уровня 5: Проведение экспертизы и оппонирование

**Длительность:** 32 ак. ч
**Формат:** письменно + защита перед комиссией.

## Сценарий

Объект — капремонт школы №47 (тот же сквозной кейс).

Получен **полный комплект сметной документации** (с уровня 4). В этот комплект **сознательно внесены 12 ошибок** (см. `02-сквозной-кейс/уровень-5-экспертиза.md`).

**Ваша роль:** внутренний эксперт. Провести экспертизу, найти ошибки, оформить заключение, защитить позицию.

## Часть А. Экспертиза (16 ч)

### Шаги

1. **Изучение комплекта** (2 ч): прочитать всю документацию, понять параметры (базисный, индексы, типы работ).

2. **Чек-лист 50 пунктов** (8 ч): пройти комплект по чек-листу, записать **все** несоответствия.

3. **Проверка ВОР** (2 ч): по выборке 10 позиций пересчитать объёмы по чертежам.

4. **Проверка ВР** (2 ч): топ-20 самых дорогих ресурсов проверить особо.

5. **Аналитическая проверка** (2 ч):
   - Стоимость м²
   - Структура затрат
   - Соотношение разделов

### Что сдаёте

- Заполненный чек-лист
- Список всех найденных несоответствий с привязкой
- Расчёт финансового эффекта по существенным
- **Экспертное заключение** по структуре:
  - Реквизиты
  - Перечень документов
  - Применённая нормативка
  - Замечания (по категориям)
  - Финансовый эффект
  - Выводы

### Цель

Найти **минимум 10 из 12** заложенных ошибок. Бонус — найденные сверх 12 (если в комплекте есть реальные неточности).

## Часть Б. Оппонирование (8 ч)

После заключения куратор играет роль **проектировщика-автора**. Получает заключение. Готовит ответы:

- По **2 из 12** ваших замечаний — обоснованный ответ "вы неправы, и вот почему"
- По **3 из 12** — попытка оппонирования (проектировщик пытается отстоять, но эксперт прав)
- Остальные — принимаются

### Шаги

1. Получить ответы проектировщика.
2. По каждому ответу:
   - **Корректные** (где эксперт ошибся) → принять, написать "Замечание снимаю"
   - **Попытки** (где эксперт прав) → удержать с дополнительным обоснованием
3. Оформить **итоговый протокол разногласий**.

### Цель

- Снять 2 правильных ответа проектировщика без сопротивления
- Удержать 3 попытки оппонирования с обоснованиями со ссылками на нормативку
- Корректный тон в формулировках

## Часть В. Публичная защита перед комиссией (8 ч)

### Состав комиссии

3 человека:
- Преподаватель курса
- Практик-сметчик из реальной организации
- Представитель экспертного сообщества (с опытом госэкспертизы)

### Процедура

1. **Презентация** (15 мин): обучающийся представляет своё заключение, демонстрирует ключевые находки и итоги оппонирования.

2. **Вопросы комиссии** (45 мин): каждый член комиссии задаёт по 3-5 вопросов:
   - По методологии (как пришли к выводу)
   - По нормативке (на какой пункт опирались)
   - По спорным позициям (могли ли решить иначе)
   - По этике (как взаимодействовали с "проектировщиком")

3. **Совещание** (15 мин): обучающийся выходит, комиссия обсуждает, выставляет коллегиальную оценку.

4. **Объявление результата**.

## Критерии оценки

| Критерий | Балл |
|---|---|
| **Часть А.** Найдено ≥ 10 из 12 ошибок | 25 |
| Корректная классификация (формальные/методические/существенные) | 10 |
| Финансовый эффект существенных замечаний — посчитан верно | 10 |
| Заключение оформлено по структуре | 10 |
| **Часть Б.** Принят правильный ответ проектировщика | 5 |
| Удержана позиция там, где эксперт прав, с нормативкой | 15 |
| Корректный тон формулировок | 5 |
| **Часть В.** Презентация — ясная, структурированная | 5 |
| Ответы на вопросы — со ссылками на нормативку | 10 |
| Спокойствие и аргументированность под давлением | 5 |
| **Итого** | **100** |

**Зачёт:** ≥ 80 баллов.

## После зачёта

**Сертификат:** "Внутренний эксперт по сметному делу в РК" — финальный сертификат курса.

При прохождении всех 5 уровней — комплексный сертификат.

## Что НЕ даёт сертификат

- Не делает выпускника **государственным экспертом** (этот статус устанавливается отдельно)
- Не заменяет специализацию по сложным типам объектов (АЭС, метро, мосты — отдельные курсы)
- Не отменяет необходимость практики 2-3 года под наставничеством для накопления "чувства нормы"

Сертификат подтверждает **методическую готовность** обучающегося работать внутренним экспертом и качественно проверять сметы.
