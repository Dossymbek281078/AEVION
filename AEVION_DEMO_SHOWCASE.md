# AEVION — Демонстрация экосистемы: все приложения, цели и связи

**Дата:** 24 марта 2026  
**Назначение:** единый документ для инвесторов, партнёров, новых участников команды и ИИ-агентов — **что такое AEVION**, какова общая цель, что делает каждый из 27+ модулей и как они связаны через Planet и Globus.

---

## Общая цель проекта AEVION

**AEVION — единая операционная система доверия для цифрового контента и IP.**

В мире, где ИИ генерирует миллиарды единиц контента ежедневно, исчезает граница между оригинальным творчеством и копией. AEVION строит инфраструктуру, в которой:

- **Авторство доказуемо** — через реестр, подпись и сертификаты.
- **Качество измеримо** — через прозрачное голосование и compliance.
- **Экосистема масштабируется** — через единую платформу для 27 продуктов с общей идентичностью, подписью и API.

**Северная звезда:** довести стоимость экосистемы к порядку 1 млрд USD через доказуемые метрики участия, выручки и партнёрств.

## Архитектура: три слоя экосистемы

```
┌─────────────────────────────────────────────────────────┐
│                    GLOBUS (карта)                        │
│  Визуальная 3D-карта всех 27 узлов • точка входа        │
├─────────────────────────────────────────────────────────┤
│              PLANET (доверие и compliance)                │
│  Артефакты • валидаторы • сертификаты • голосование      │
│  Премии Music Awards / Film Awards • Merkle snapshots   │
├─────────────────────────────────────────────────────────┤
│           ПРОДУКТОВЫЕ МОДУЛИ (27 узлов)                   │
│  Auth • QRight • QSign • Bureau • QTrade • QPayNet ...  │
│  HealthAI • Kids AI • CyberChess • LifeBox • и др.     │
└─────────────────────────────────────────────────────────┘
```

**Принцип:** каждый модуль подключает общий Auth, общий API и общий Planet — не строит всё с нуля. Один новый модуль = только доменная логика.

---

## PLANET — сквозной слой доверия

**Что это:** слой доказуемого соответствия и репутации. Автор подаёт работу (код, веб, аудио, видео) → система фиксирует хеши и запускает валидаторы → выдаёт сертификат → участники Planet голосуют по категориям.

**Ключевые возможности:**
- Evidence root и цепочка валидаторов для каждого артефакта
- Публичная карточка работы с compliance summary
- Голосование по номинациям (1-5, по categoryId)
- Merkle snapshots для аудита голосов
- Метрика Y = участники с активным CodeSymbol
- Две витрины премий: Music Awards и Film Awards

**Маршруты:** `/planet`, `/planet/artifact/[id]`, `/awards`, `/awards/music`, `/awards/film`

**API:** `GET /api/planet/stats`, `POST /api/planet/submissions`, `GET /api/planet/artifacts/:id/public`, `POST /api/planet/artifacts/:id/vote`

## GLOBUS — визуальная карта экосистемы

**Что это:** интерактивный 3D-глобус с 27 узлами-маркерами. Каждый узел = один продукт AEVION с координатами, статусом и ссылкой.

**Цель:** одним взглядом показать масштаб, зрелость и связность экосистемы. Для инвестора — живой пульс портфеля. Для пользователя — единая точка входа.

**Маршрут:** `/` (главная страница)

## 27 модулей: цель каждого

### Фаза A — Ядро (уже работает или близко к MVP)

#### 1. GLOBUS
**Цель:** единая карта экосистемы и точка входа.  
**Что делает:** 3D-визуализация 27 направлений с маркерами, статусами и ссылками. Клик по узлу ведёт в осмысленный сценарий.  
**Ценность для инвестора:** один URL показывает масштаб и зрелость всей матрицы продуктов.

#### 2. QRight
**Цель:** независимый реестр цифровых объектов и приоритетов авторства.  
**Что делает:** фиксация факта существования идеи/контента с хешем, метаданными и привязкой к владельцу. База для QSign и Bureau.  
**Ценность:** строит юридически осязаемый слой поверх цифрового творчества.

#### 3. QSign
**Цель:** криптографическая целостность артефактов для всей экосистемы.  
**Что делает:** HMAC-подпись и проверка payload'ов. Один формат для всех модулей — что подписано, то неизменно.  
**Ценность:** опора для compliance и аудита.

#### 4. IP Bureau (aevion-ip-bureau)
**Цель:** электронное патентное бюро — от реестра к сертификату готовности.  
**Что делает:** связывает QRight и QSign в сценарий, понятный юристу и инвестору. Локальное хранение подписей + путь к экспорту.  
**Ценность:** ускоряет due diligence — артефакты собраны в одном контуре.

#### 5. Auth
**Цель:** единая идентичность пользователя во всей экосистеме.  
**Что делает:** JWT-аутентификация, токен для всех Q-продуктов и Planet.

---

### Фаза B — ИИ и мультичат

#### 6. QCoreAI
**Цель:** общий ИИ-движок для всех Q-продуктов.  
**Что делает:** единый контракт API агентов и оркестрации. Централизованные политики и лимиты.  
**Ценность:** экономия OPEX на ML/ops и предсказуемый unit economics по токенам.

#### 7. Multichat Engine
**Цель:** мультичат с параллельными агентами под разные задачи.  
**Что делает:** пользователь ведёт код, финансы, IP и контент в одном интерфейсе. Агенты изолированы по контексту.  
**Ценность:** готовая основа для B2B white-label ассистентов.

#### 8. QFusionAI
**Цель:** гибридный движок под разные провайдеры моделей.  
**Что делает:** переключение моделей без смены UX. Автоподбор модели под задачу — баланс цены и качества.  
**Ценность:** снижение vendor lock-in, устойчивость к изменениям рынка LLM.

---

### Фаза C — Финтех

#### 9. QTradeOffline
**Цель:** торговля и переводы без постоянного интернета.  
**Что делает:** офлайн-транзакции с последующей синхронизацией. Расширение TAM на регионы с нестабильной связью.  
**Ценность:** дифференциация на рынке финтех-приложений.

#### 10. QPayNet Embedded
**Цель:** встроенное платёжное ядро для приложений AEVION.  
**Что делает:** единые счета, комиссии и учёт внутри экосистемы. Быстрая монетизация новых модулей.  
**Ценность:** контроль рисков и аудит в одном контуре.

#### 11. QMaskCard
**Цель:** карта с маскированием и антифродом.  
**Что делает:** снижение chargeback и утечек реквизитов. Премиальный B2C-опыт.  
**Ценность:** cross-sell с QPayNet и QTrade.

---

### Фаза D — Здоровье и благополучие

#### 12. HealthAI
**Цель:** персональный AI-коуч здоровья (не замена врачу).  
**Что делает:** sticky daily use, данные для премиум-подписок. Кросс-продажи с QGood.  
**Ценность:** удержание пользователя в экосистеме.

#### 13. QLife
**Цель:** долголетие и антистарение на основе биомаркеров.  
**Что делает:** планы по данным здоровья, премиальный сегмент с высоким LTV.  
**Ценность:** долгосрочные горизонты удержания данных для R&D-партнёров.

#### 14. QGood
**Цель:** психологическое благополучие с ИИ и офлайн-режимом.  
**Что делает:** персонализированная поддержка, ESG-повестка.  
**Ценность:** социально значимый продукт — барьер для копирования через глубокую персонализацию.

#### 15. PsyApp (psyapp-deps)
**Цель:** поддержка выхода из зависимостей.  
**Что делает:** долгие программы удержания, партнёрства с НКО и клиниками.  
**Ценность:** доверие через приватность и этичный дизайн.

### Фаза D — Контент и дети

#### 16. Kids AI Content
**Цель:** детское обучение, многоязычие, безопасный контент.  
**Что делает:** родительский сегмент с высокой готовностью платить. Локализация как масштабирование.  
**Ценность:** доверие бренда AEVION в семье — апселл на другие сервисы.

**Цель:** цифровой аватар для работы и общения.  
**Что делает:** новый класс подписок через кастомизацию. Мост между человеком и всеми Q-сервисами.  
**Ценность:** интеграция с Multichat и контентом.

#### 18. Voice of Earth
**Цель:** международный музыкальный контент-проект.  
**Что делает:** эмоциональный маркетинг, кросс-медиа (музыка → приложения → мерч).  
**Ценность:** культурный ESG-нарратив и узнаваемость бренда.

## Продуктовые витрины Awards

### AEVION Music Awards
- Аналог музыкальной премии для ИИ и цифровой музыки
- Номинации: лучшая запись, саунд-продакшн, ИИ-творчество, текст/концепция
- Подача через Planet с `type=music`, `productKey=aevion_award_music_v1`
- Маршрут: `/awards/music`

### AEVION Film Awards
- Аналог кинопремии для ИИ и цифрового кино
- Номинации: режиссура, сценарий, монтаж, ИИ-творчество
- Подача через Planet с `type=movie`, `productKey=aevion_award_film_v1`
- Маршрут: `/awards/film`

### Лидерборд
- Сортировка: по среднему баллу, по числу голосов, по дате
- Coverage: % голосовавших от Y (участники с активным CodeSymbol)
- Медали 🥇🥈🥉 для топ-3
- Micro-bar визуализация покрытия

## Как запустить для демонстрации

```bash
# Клонировать
git clone <repo> aevion-core && cd aevion-core

# Backend
cd aevion-globus-backend
cp .env.example .env          # настроить DATABASE_URL
npm install && npx prisma migrate deploy && npm run build && npm start

# Frontend (отдельный терминал)
cd frontend
cp .env.example .env.local    # NEXT_PUBLIC_API_BASE_URL=/api-backend
npm install && npm run build && npm start

# Или dev-режим из корня:
npm run dev
```

**Маршрут демо для инвестора:**
1. `/` — Globus (масштаб экосистемы)
2. `/demo` — спектакль: все модули с выгодами
3. `/awards` — премии (живые метрики)
4. `/planet` — подача артефакта
5. `/planet/artifact/[id]` — публичная карточка + голосование
