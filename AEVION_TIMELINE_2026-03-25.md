## Что ускоряет

1. **Второй разработчик** — на параллельный трек (финтех или ИИ): сдвигает Волну 2-3 на 1-2 месяца раньше
2. **Дизайнер** — снимает UX-блокеры и ускоряет «полировку» для демо
3. **LLM API ключи** — без них QCoreAI/Multichat невозможны; нужны до Волны 3
4. **Postgres на хостинге** — Railway/Render, нужен для QTrade persistence

## Что замедляет

1. **Распыление на все 27 сразу** — вместо E2E срезов → +6-12 месяцев
2. **Отсутствие deploy** — без публичного URL нет фидбека → +2-3 месяца на итерации
3. **Регуляторика финтеха** — QPayNet/QMaskCard требуют юридической экспертизы
4. **Смена приоритетов** — каждый «давай ещё вот это» без завершения текущего = +40% к срокам
