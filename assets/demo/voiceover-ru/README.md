## Как собрать видео

### DaVinci Resolve (бесплатный, рекомендую)

1. New Project → 1920×1080 / 30fps.
2. Импорт screencast (видео-дорожка) + все 9 mp3 (аудио-дорожки).
3. На таймлайне:
   - `01-opening.mp3` ставится в начало (0:00).
   - Между секциями — паузы для визуальных действий по таймкодам из скрипта.
   - Если voiceover короче таймкод-слота — оставь тишину (зритель смотрит как ты кликаешь).
   - Если длиннее — слегка уплотни клики или растяни паузу.
4. Экспорт: H.264 / 1080p / 30fps / 8-12 Mbps, audio AAC 192 kbps.

### Loom + Audacity (быстрая альтернатива)

1. Audacity: open all 9 mp3 → tracks → arrange по таймкодам → mixdown → export wav.
2. Loom screencast (без голоса) → upload to YouTube как unlisted.
3. YouTube Studio → Edit Audio → upload wav → align.

### Тестовая длина

Голосовых дорожек в сумме ~2:30-3:00. Остальные ~5 минут — это
визуальные действия (загрузка файла, клики, ожидание Stripe). Целевая
**итоговая длина видео — 7:30-8:30**.

## Перегенерировать секцию

Если переписал какой-то кусок скрипта — открой `voiceover-ru/`,
удали соответствующий mp3, и прогони ту же команду что и в первый раз
(см. ниже). Сгенерируется только удалённая секция, если добавить проверку
`if (Test-Path $outPath) { continue }`.

```powershell
# Полная регенерация всех секций — повторить блок из чата
# с новыми текстами в массиве $sections
$key = "<elevenlabs api key — text-to-speech access>"
$voiceId = "5ulR11sYgBk4l61h0VIu"
# ... (см. полный код в истории сессии 2026-05-06)
```
