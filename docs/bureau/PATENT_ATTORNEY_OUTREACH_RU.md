## 1. Кто мы

AEVION Digital IP Bureau — цифровая платформа фиксации
интеллектуальной собственности. К моменту обращения к патентному
поверенному наш клиент уже:

- Зарегистрировал произведение в QRight (SHA-256 hash + Ed25519 +
  Shamir 2-of-3 + Bitcoin-anchored timestamp через OpenTimestamps).
- Прошёл KYC-проверку личности через сертифицированного провайдера
  (Sumsub / Veriff). У нас на руках — verified имя, страна, тип
  документа.
- Имеет публично верифицируемый URL сертификата + PDF + офлайн-бандл
  JSON, в котором уже есть всё для доказательства приоритета.

То есть **на момент перехода к Вам клиент уже полностью
идентифицирован**, имеет фиксированный приоритет (Bitcoin-таймстамп),
и располагает упорядоченными метаданными произведения (название, тип,
описание, страна). Это убирает 60–80% рутины на этапе формирования
заявки.

## 2. Что мы предлагаем

**Partnership-канал:** AEVION направляет клиента, готового подать
заявку в патентное ведомство, к Вам. Мы обеспечиваем:

- **Структурированный пакет заявки** в открытом формате (JSON +
  готовый PDF для нотариального удостоверения, если требуется).
- **Доказательство приоритета** через AEVION-сертификат — снимает
  необходимость отдельной публикации даты приоритета.
- **Разовая или абонентская монетизация:** клиент оплачивает AEVION
  $299–$1500 за tier Filed (зависит от юрисдикции и сложности),
  AEVION удерживает комиссию платформы 15–25% и перечисляет остаток
  Вашему бюро вместе с государственными пошлинами.
- **Поток клиентов:** на основе наших текущих темпов мы прогнозируем
  10–30 заявок в месяц на tier Filed в первые 12 месяцев.

**Юрисдикции, по которым ищем партнёров:**

- Казахстан — НИИС МЮ РК / Казпатент (приоритет — у нас базовая
  юрисдикция).
- Россия — Роспатент.
- ЕС — EPO (European Patent Office).
- США — USPTO.
- WIPO PCT — для международной защиты.

Готовы работать с одним бюро по нескольким юрисдикциям или с
несколькими бюро по одной каждой.

## 3. Технический интерфейс

Мы предоставим Вам:

- **Защищённый кабинет партнёра** (`/bureau/partners/<slug>`):
  - Лента входящих заявок с метаданными от AEVION-сертификата.
  - API endpoint для подтверждения принятия заявки в работу.
  - API endpoint для обновления статуса (filed → published →
    granted / rejected) с привязкой к AEVION-сертификату.
- **Webhook-уведомления** о новых заявках в реальном времени.
- **Стандартизированный JSON-формат заявки** (мы используем формат,
  совместимый с PCT-схемой WIPO ST.36 для будущей электронной подачи
  через ePCT).
- **Брендинг:** AEVION упоминает Вас как партнёра по `<юрисдикция>`
  в публичном каталоге `/bureau/partners`, ссылается на Ваш сайт.

## 4. Юридические рамки

- **Партнёрский договор** — типовая форма с разделами: объём услуг,
  тарифы, обработка ПДн, конфиденциальность переданных материалов,
  ответственность, расчёты, разрешение споров.
- **Ответственность AEVION** — ограничена корректной передачей
  данных клиента; качество подачи заявки и результат рассмотрения
  ведомством — Ваша зона ответственности.
- **NDA** покрывает все материалы клиента, переданные через AEVION.
