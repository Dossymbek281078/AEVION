# Методичка уровня 3 "ПТО"

**Длительность:** 64 ч (16 теории + 32 практики + 16 на кейс)
**Цель:** обучающийся ведёт сметную работу на действующем объекте: исполнительные сметы, КС-2/КС-3, корректировки, дополнительные работы.
