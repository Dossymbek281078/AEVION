# Урок 2. Методология сметной экспертизы

**Длительность:** 4 ак. ч (2 теории + 2 практики)

## Цель

После урока обучающийся:
- Применяет 9-слойную методологию проверки сметы
- Использует чек-лист на 50 пунктов
- Систематизирует найденные несоответствия

## Теория (90 мин)

### 2.1. Принцип слоёв

Эксперт смотрит смету **по слоям** — от формальных к сущностным. Это:
- Систематизирует процесс
- Не даёт пропустить пункты
- Позволяет рано остановиться при критическом нарушении (например, нет редакции СН РК — дальше можно не смотреть)

### 2.2. 9 слоёв проверки

#### Слой 1. Комплектность
- Все ли необходимые документы присутствуют (ЛСР, ОС, ССР, ВОР, ВР, ПЗ)
- Все подписаны
- Базовое оформление в порядке

**Цель слоя:** убедиться, что есть **что проверять**.

#### Слой 2. Нормативная база
- Применённая редакция СНБ — актуальна
- Применённые СН РК — актуальны
- Индексы пересчёта — на нужный квартал
- Редакция указана в ПЗ

**Цель слоя:** убедиться, что проверка пойдёт по **правильным правилам**.

#### Слой 3. Соответствие проекту
- Каждая позиция в смете подтверждается проектным решением
- Технологии работ соответствуют проекту
- Материалы соответствуют спецификациям
- Объёмы соответствуют чертежам

**Цель слоя:** проверить, что **состав** работ адекватен проекту.

#### Слой 4. Корректность объёмов
- Выборочная проверка по чертежам
- Проёмы вычтены
- Откосы посчитаны отдельно
- Нет дублирования между позициями
- Округления корректны

**Цель слоя:** проверить **числа** в графе "объём".

#### Слой 5. Корректность расценок
- Применены наиболее точные нормы
- Единицы измерения правильные
- Шифры из актуальных сборников
- Нет необоснованного применения "по аналогии"
- Нет применения более дорогой нормы при наличии точной

**Цель слоя:** проверить **выбор** расценок.

#### Слой 6. Ресурсы и индивидуальные цены
- Базисные цены — из актуального сборника
- Индивидуальные — с 3 КП и обоснованием
- К индивидуальным не применился индекс
- Транспортные расходы учтены корректно

**Цель слоя:** проверить **цены ресурсов**.

#### Слой 7. Накладные и прибыль
- Норматив НР для вида работ корректный
- Норматив СП корректный
- База начисления (ФОТ / ПЗ) корректная
- Понижающие коэффициенты применены с обоснованием

**Цель слоя:** проверить **накрутки**.

#### Слой 8. Лимитированные затраты
- Временные не дублируются с ЛСР
- Зимнее удорожание — только на чувствительные работы и в зимние месяцы
- Непредвиденные — норматив для типа объекта
- Содержание заказчика — по СН РК 8.02-05
- АН — корректно

**Цель слоя:** проверить **главу 8, 9, 10, 12 ССР**.

#### Слой 9. Арифметика и формы
- Итоги по ЛСР сходятся
- Итоги ОС = сумма ЛСР
- Итоги ССР = сумма ОС + лимитированные
- НДС применён к итогу с резервом
- Все формы (1, 2, 3, ВОР, ВР) выгружены и читаемы

**Цель слоя:** убедиться, что **математика** правильная.

### 2.3. Чек-лист на 50 пунктов

Полный чек-лист — в файле [`03-lms/03-чек-листы.md`](../03-lms/03-чек-листы.md), раздел "Чек-лист эксперта".

Эксперт идёт по этому списку **строго по порядку**. Каждый пункт даёт оценку:
- ✅ соответствует
- ⚠️ требует уточнения (просим обоснование)
- ❌ не соответствует (замечание)

### 2.4. Классификация замечаний

После прохождения по слоям — **классифицировать** найденные несоответствия:

#### Формальные
- Опечатки, отсутствие подписи, неверная нумерация форм
- **Влияние:** низкое; исправляются за 1 итерацию
- **Срок устранения:** 1-3 дня

#### Методические
- Неверный норматив НР
- Неверный способ применения индекса
- Использование расценки не по назначению
- **Влияние:** среднее; могут изменить итог на 2-10%
- **Срок устранения:** 5-10 дней

#### Существенные
- Двойной счёт
- Завышение объёмов (≥ 5%)
- Индивидуальные цены без обоснования
- Лишние работы
- **Влияние:** высокое; могут изменить итог на 10%+
- **Срок устранения:** 10-30 дней (требуется пересчёт)

### 2.5. Финансовый эффект замечаний

Эксперт оценивает не только "правильно/неправильно", но и **в тенге**:
- Сколько было в смете
- Сколько должно быть после устранения
- **Разница** — финансовый эффект

Это:
- Помогает заказчику оценить качество эксперта
- Создаёт прецедент при споре с проектировщиком
- Является доказательством при защите бюджетных средств

### 2.6. Структура экспертного заключения

```
1. Реквизиты заключения
   - № заключения, дата
   - Объект
   - Заказчик
   - Проектная организация
   - Эксперт ФИО

2. Перечень рассмотренных документов
   (с указанием объёма и редакции)

3. Применённая нормативная база
   (что использовал эксперт для сверки)

4. ЗАМЕЧАНИЯ
   4.1. Существенные (с финансовым эффектом)
   4.2. Методические
   4.3. Формальные

5. ВЫВОДЫ
   - Положительное / отрицательное / условное
   - Сроки устранения
   - Финансовый эффект (если применимо)

6. ПОДПИСИ
```
