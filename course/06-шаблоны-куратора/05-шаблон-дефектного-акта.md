## Описание выявленных дефектов / необходимости дополнительных работ

### 1. Обнаружено

При выполнении работ на объекте по договору подряда комиссия обнаружила следующее:

```
[ОПИСАНИЕ ДЕФЕКТОВ / НЕОБХОДИМОСТИ ДОПОЛНИТЕЛЬНЫХ РАБОТ]

Например:
"При вскрытии существующего пола в помещениях №№ 1.01, 1.02, 1.03, 1.04
обнаружено повреждение деревянных лаг основания вследствие гниения.
Лаги в количестве ~28 пог.м требуют замены. Дополнительно — устройство
антисептирования примыкающих участков и усиление металлическим швеллером
по периметру (~24 пог.м)."
```

## Причина возникновения

```
[АНАЛИЗ ПРИЧИН]

Например:
"Повреждения вызваны воздействием влаги в подпольном пространстве,
накопившейся за период эксплуатации здания (более 30 лет). Указанные
дефекты не были видны при разработке проектной документации, так как
вскрытие пола не предусматривалось проектом обследования."
```

---

## Предлагаемое решение

```
[ПЕРЕЧЕНЬ РАБОТ ДЛЯ УСТРАНЕНИЯ]

1. Демонтаж и вывоз повреждённых лаг — 28 пог.м
2. Антисептическая обработка прилегающих деревянных элементов
3. Устройство нового основания: лаги пропитанные — 28 пог.м
4. Усиление по периметру швеллером 80×40 — 24 пог.м
5. Восстановительные работы по полу
```

---

## Заключение комиссии

Комиссия считает:

- ☐ **Дополнительные работы необходимы** для возможности продолжения строительства
- ☐ Перечисленные работы **не были предусмотрены** в первоначальной проектной документации и в смете
- ☐ Требуется **дополнительное соглашение** к договору подряда с включением дополнительных работ

**Рекомендация:** Заключить дополнительное соглашение к договору подряда с включением дополнительных работ согласно прилагаемой дополнительной ЛСР.
